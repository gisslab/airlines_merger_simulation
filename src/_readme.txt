# src/ Folder

This folder contains the core code for downloading, cleaning, and processing airline data for the merger simulation project.

## CODE FILES:
- `data_utils.py`: Contains utility functions for data handling, including fetching and processing economic data
- `download_db1b.py`: Script to download and process DB1B market data
- `processing_data.do`: Stata script for processing data related to the airline merger simulation
- `_readme.txt`: Provides an overview of the src/ folder structure

## RUNNING ORDER:
1. Run data downloads .py files, see instructions (Run: download_db1b.py)
2. Preprocess downloaded data: processing_data.do
3. Demand estimation → TODO
4. Merger simulation → TODO

For detailed information about specific files, please refer to inline documentation.
