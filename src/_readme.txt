# Airlines Merger Simulation Code
=================================

The code folder contains the following files: 
- `src/data_utils.py`: Contains utility functions for data handling, including fetching and processing economic data.
- `src/download_db1b.py`: Script to download and process DB1B market data
- `src/_readme.txt`: Provides an overview of the project and its structure.
- `processing_data.do`: Stata script for processing data related to the airline merger simulation.
